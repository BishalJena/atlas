# Changelog

All notable changes to AIDP Compute Arena will be documented in this file.

## [0.1.0] - 2025-12-30

### Added
- **Dashboard Layout** — Main page with responsive grid layout
- **GPU Map** — Interactive world map with animated node markers
- **Node Selector** — GPU cards showing specs, pricing, live metrics
- **Workload Picker** — LLM Chat, Image Generation, Video Processing options
- **Metrics Panel** — Real-time VRAM, latency, cost, temperature display
- **Prompt Input** — Text input with sample prompts and loading states
- **Output Display** — Typewriter effect for LLM, image/video viewer
- **Session Stats** — Aggregate job count, cost, tokens tracker
- **Header** — App header with network status and wallet connect button
- **Design System** — Dark theme, glassmorphism, animations in globals.css
- **Mock Data** — 5 global GPU nodes with realistic specs

### Technical
- Next.js 16 with App Router
- TypeScript for type safety
- Tailwind CSS for styling
- React hooks for state management
