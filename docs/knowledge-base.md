## â“ FAQ

**What is AIDP?**
> AIDP is a decentralized AI and GPU compute network built as a DePIN, enabling permissionless access to AI tools and on-demand GPU resources.

**What problem does AIDP solve?**
> AIDP reduces dependence on centralized cloud providers by enabling a decentralized marketplace for GPU compute, coordinated transparently through Web3 infrastructure.

**Who can use AIDP?**
> Developers, Web3 projects, AI builders, startups, and GPU node operators can all participate in the AIDP ecosystem.

**How does the DePIN model work on AIDP?**
> Independent GPU providers contribute hardware to the network, while users consume compute. Resource allocation and rewards are coordinated in a trust-minimized manner.

**How do GPU providers earn rewards?**
> GPU providers earn rewards based on compute usage, uptime, and network demand, aligned through protocol-level incentives.

**Is AIDP non-custodial?**
> Yes. AIDP is designed to be non-custodial, allowing users to retain control over their assets and infrastructure.
